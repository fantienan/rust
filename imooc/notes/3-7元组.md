# 元组

元组是将多个具有各种类型的值组合成一个复合类型的通用方法. 元组有固定的长度: 一旦声明, 它们的大小就不能增长或收缩.

我们通过在括号内写一个逗号分隔的值列表来创建一个元组. 元组中的每个位置都有一个类型, 元组中不同值的类型不必相同.
```rust
fn main() {
    let a: i32 = 10;
    let b: char = 'A';

    // 创建一个元组
    let mytuple: (i32, char) = (a, b);

    // 从元组中读取一个值
    println!(".0={:?}", mytuple.0);
    println!(".1={:?}", mytuple.1);

    // 解封装
    let (c, d) = mytuple;
    println!("c={} d={}", c, d);
}
```
